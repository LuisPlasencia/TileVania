# TileVania
Unity 3.1.2 2d Platform Game
- WASD or Arrow keys to move
- Space to jump
- Left Click to shoot

Playable builds: https://drive.google.com/drive/folders/14_67Wq4b1otDy_ixzzOd_BPq3h8Ndz2N?usp=sharing
